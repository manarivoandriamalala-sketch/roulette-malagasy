# 🎰 Roulette Malagasy

Une roulette européenne développée en HTML, CSS et JavaScript.

## Fonctionnalités

- Mise sur les numéros
- Rouge / Noir
- Pair / Impair
- Animation de roulette
- Sauvegarde du crédit
- Historique des tirages
- Système de recharge

## Technologies

- HTML5
- CSS3
- JavaScript
- Canvas API
- LocalStorage
